# Relaxicons

Relaxicons is a CLI that turns any Iconify icon (e.g. `lucide:home`) into a polished component for React, Vue, Angular, or Laravel.

```bash
npm install -g relaxicons
relaxicons init
relaxicons add lucide:home
```

<CopyButton value="relaxicons add lucide:star" label="Copy Example" />

## Features

- Fetch & suggest icons from collections
- Transform SVG (strip dimensions, set `currentColor`)
- Output multi-framework components
- Auto formatting & barrel export updates
- Fuzzy suggestions & list filtering
